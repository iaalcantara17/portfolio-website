Fortran95 Compiler Project

This project combines a lexical analyzer, a recursive-descent parser, and an interpreter for the SFort95 language. The C++ implementation handled state-based transitions, error detection, and run-time checks. The key components include:

  • Lexical analysis to recognize multi-character tokens like identifiers and constants.
  • Parsing and validating expressions using recursive-descent parsing, including operator precedence.
  • Building a symbol table to track variable types and perform type checks during interpretation.
  • Error reporting for common issues such as missing semicolons, undefined variables, and division by zero.
